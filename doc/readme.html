Voice Speed Manager

An NVDA add-on to automatically adjust speech speed based on language and switch voice/language based on the active application.

Features

  • Language-Specific Speed: Set different speech rates for different languages (e.g., German at 40%, English at 20%).
  • Application-Specific Voice: Automatically switch the synthesizer language when focusing specific applications (e.g., switch to German for Outlook).

Configuration

Go to NVDA Menu -> Preferences -> Settings -> Voice Speed Manager.

Application Rules

Add rules to map an application (e.g., outlook.exe) to a specific language code (e.g., de for German, en for English).

Language Rates

Set the desired speech rate (0-100) for each language code. The add-on will automatically apply this rate whenever the synthesizer switches to that language.

Usage

Once configured, simply use your computer as normal. When you switch to an application with a configured rule, the voice will change automatically. If the new voice's language has a specific rate set, the speech speed will also adjust.
